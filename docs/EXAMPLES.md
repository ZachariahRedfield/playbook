# Playbook CLI Examples

## Analyze a repository

```bash
playbook analyze
```

## Verify repository governance

```bash
playbook verify
```

## Initialize Playbook

```bash
playbook init
```

## Sample CLI output

```text
$ playbook analyze
Detected Stack

Framework: Next.js
Database: Supabase
Styling: Tailwind

$ playbook verify
PASS  requireNotesOnChanges
All governance checks passed.
```

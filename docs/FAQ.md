# Playbook FAQ

## What does Playbook do?

Playbook analyzes repositories and enforces deterministic engineering governance for architecture, documentation, and repository standards.

## Is Playbook only for AI agents?

No. Playbook is useful for both human contributors and AI coding agents working in modern repositories.

## Does Playbook modify my code?

Playbook governance checks validate repository rules; they do not rewrite your source code during verification.

## Does Playbook require a specific framework?

No. Playbook is framework-agnostic and focuses on repository governance rules.

## Can Playbook be used with any repository?

Yes. Playbook is designed to work across repositories that need consistent architecture and documentation governance.

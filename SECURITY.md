# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately to repository maintainers. Include:

- affected package(s) or file path(s)
- reproduction steps
- impact assessment
- suggested remediation (if available)

Do not open public issues for unpatched vulnerabilities.

## Supported Versions

Playbook is currently pre-1.0. Security fixes are applied to the latest development line.

## Scope

This policy covers the CLI, governance engine, templates, and repository automation in this project.

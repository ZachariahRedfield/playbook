# Numeric Cosmology Pattern — Sevenfold Plus

## Rule
0, 8, and 9 are meta-states around the seven, not extra entries beside them.

## Pattern
- 0 = substrate
- 1–7 = differentiation
- 8 = synthesis
- 9 = persistence

## Layer Mapping Requirement
Every symbolic number must map across:
- metaphysical layer
- psychological layer
- aesthetic layer
- mechanical layer
- historical layer

## Practical Mapping Template
For each symbol entry, require:
1. Cosmological function
2. Character wound/virtue arc
3. Visual + audio motif stack
4. Encounter/mechanics behavior
5. World-history persistence impact

## Failure Modes
- Overfitting correspondences across too many traditions makes the system feel arbitrary.
- Letting dev mnemonics leak into canon reduces originality.
- Treating 8 as another realm instead of a reconciliation layer weakens the ontology.
- Treating 9 as transcendence instead of legacy/consequence duplicates 8.
